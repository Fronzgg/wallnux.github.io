## Как протестировать:

### 1. Добавить бейджи пользователю:
```bash
node create-devwallnux.js
```

Или через админ-панель:
1. Войти как администратор
2. Открыть админ-панель (кнопка в левом меню)
3. Выбрать пользователя
4. Добавить бейджи: verified, team, developer

### 2. Проверить отображение:

#### В списке ЛС:
1. Открыть список Direct Messages
2. Найти пользователя с бейджем verified или team
3. Должна быть синяя галочка ✓ рядом с именем

#### В профиле:
1. Кликнуть на аватар пользователя
2. Откроется профиль
3. Рядом с именем должна быть синяя галочка ✓
4. Ниже должны быть карточки с бейджами

#### В сообщениях:
1. Открыть чат с пользователем
2. Рядом с именем автора должна быть галочка ✓
3. При наведении показывается "Официальный аккаунт"

### 3. Проверить бейдж разработчика:
- В профиле должен отображаться как **</>**
- При наведении: "Разработчик"
