# 🔓 Как открыть DevTools

## Секретная комбинация:
**Ctrl + Alt + Shift + D**

## Что происходит:
1. Нажми Ctrl + Alt + Shift + D
2. Появится уведомление "🔓 Защита DevTools отключена"
3. Теперь можешь открыть DevTools (F12)
4. Правая кнопка мыши работает
5. Консоль работает

## Чтобы включить обратно:
Снова нажми **Ctrl + Alt + Shift + D**

## Что работает когда защита отключена:
✅ F12 - открыть DevTools
✅ Ctrl+Shift+I - открыть DevTools
✅ Ctrl+Shift+J - открыть консоль
✅ Ctrl+U - посмотреть исходный код
✅ Правая кнопка мыши
✅ console.log() работает
✅ Можно тестировать мобильную версию (Ctrl+Shift+M)

## Для обычных пользователей:
Защита работает как обычно - DevTools заблокированы
